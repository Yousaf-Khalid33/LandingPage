# Muhammad Yousaf Khalid — Portfolio Landing Page

A clean, responsive personal portfolio website built with `HTML`, `CSS`, and `JavaScript`. This static site showcases professional experience, internships, education, and featured projects, including a live HireStream demo link.

## Features

- Responsive layout for desktop and mobile
- Smooth section navigation and polished UI cards
- Experience timeline and internship highlights
- Project showcase with live demo link to HireStream
- Education section emphasizing React, Node.js, and AI expertise

## Project Structure

- `index.html` — main portfolio page markup
- `style.css` — visual styling and responsive design
- `script.js` — interactive page behavior and animations
- `.gitignore` — ignores local IDE and build artifacts

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Yousaf-Khalid33/LandingPage.git
   cd LandingPage
   ```
2. Open `index.html` in a browser.

## Deployment

This project is ready to deploy as a static site.

### Deploy with Vercel

1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```
2. Deploy from the project folder:
   ```bash
   cd "c:\\Users\\PMLS\\Desktop\\Yousaf Khalid Landing Page"
   vercel
   ```
3. For a production link:
   ```bash
   vercel --prod
   ```

### GitHub + Vercel

- Push the repository to GitHub
- Connect the repo in Vercel
- Vercel will detect that this is a static site and deploy `index.html`
